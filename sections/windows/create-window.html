This module works in both the main and renderer process. The examples below show the code running the demo button which is using shell in the renderer process.

Create a new Window
Supports: Win, OS X, Linux | Process: Main

The BrowserWindow module gives you the ability to create new windows in your app. It is a main process module but can be used in the renderer process with the remote module. We'll do that in this demo.

There are a lot of options when creating a new window. A few are in this demo, but visit the documentation for the full list.
