Communication

The ipc (inter-process communication) module allows you to send and recieve synchronous and asynchronous messages between the main and renderer processes. There is a version of it available for both processes.

Pro-Tip

Use remote module to interact directly with main process

The remote module allows you to directly use main process modules from within the renderer process, bypassing the need to send and or recieve a message between the two..

Using a sheet-style dialog

  var ipc = require('electron').ipcMain;
  var dialog = require('dialog');
