Tray

The tray module in Electron allows you to create an icon in the operating system's notification area. This icon can also have a context menu attached.

This module is a main process module but can be triggered by using the ipc module to talk to that process from the renderer process.

Open the full API documentation in your browser.

Tray
Supports: Win, OS X, Linux | Process: Main

The button above sends a message to the main process using the ipc module. In the main process the app is told to place an icon in the tray for one minute. The limited time is for deomonstration purposes, you're more likely to keep the icon in the tray for as long as your app is active.
