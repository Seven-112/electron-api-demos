Pro-Tip

Electron includes verions of Chromium, Node.js and V8. You can find exactly which version by checking process.

Get there quickly by opening up developer tools in Electron and typing process.versions

Getting Process Versions in Electron
// Returns version of Chromium in use
process.versions.chrome;
// Returns version of V8 in use
process.versions.v8;
// Returns version of Node in use
process.versions.node;
