Creating Native Dialogs

The diaglog module in Electron allows you to make use of native system dialogs.

Open a File or Directory

Supports: Win, OS X, Linux

Opening a file must be done in the main process so if you're wanting to launch the dialog from a browser window (render process) you'll use the ipc module to communicate from one process to the other.
