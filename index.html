Electron Demos

What do you want to do?

Window

Create a new window.

Create a new window in your app. It can be frameless, hidden or a web view.

Manage window state.

Change the window size, location or manage the recent documents.
Menus

Create menus.

You can create a top menu as well as a context (right-click) menu for your app.
Native User Interface

Open external links or system file manager.

Launch links in the user's default web browser or show files in the OS file manager.

Use system dialogs.

Use native system windows like a file/directory chooser, file saver or an error or message dialog.

Use the tray icon.

Give your app an icon and context menu in the top menu tray.
Communication

Communicate between the two processes.

Use the `ipc` module to communicate to and from the main and renderer processes.
System

Find out app or user system info.

You can information on the version of Electron, operating system in use and the specs of the users screen.

Copy and Paste with the clipboard.

Add to or read from the user's clipboard storage.
Printing

Print to PDF.

Provide an option for printing to PDF in your app.
