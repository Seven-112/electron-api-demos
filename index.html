Electron Demos

What do you want to do?

Windows

Create a new window.

Create a new window in your app. It can be frameless, hidden or a web view.

Manage window state.

Change the window size, location or manage the recent documents.

Menus

Customize menus.

Create an application (top) menu as well as a context (right-click) menu for your app.

Native User Interface

Open external links or system file manager.

Launch links in the user's default web browser or show files in the system's file manager.

Use system dialogs.

Use native dialogs like a file or directory picker, file saver, an error or info dialog.

Put your app in the tray.

Give your app an icon and context menu in the top menu tray.

Communication

Communicate between the two processes.

Send messages and fire off actions between the main and renderer processes.

System

Get app or user system information.

Get the version of Electron, the operating system in use and the specs of the user's screen.

Copy and paste from the clipboard.

Add to or read from the user's clipboard.

Printing

Print to PDF.

Provide an option for printing to PDF in your app.
